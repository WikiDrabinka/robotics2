# Dataset 
Download [dataset](https://chmura.put.poznan.pl/s/ZQJdgHoN2b4KFTL) into directory

# Model training
Run [model_trainer.py](model_trainer.py) script. Your model will be saved to `model_final_final_02_final_please.onnx`.

# Running
Open [file.ipynb](file.ipynb) and run all the cells up to `d.run_loop()`

To stop the jetbot, run `d.stop()`
